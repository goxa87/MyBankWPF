Модель банка

реализованы технологии
*wpf
*ADO.Net
*EntityFramework


БД на базе MSSqlServer

в файле DBStart есть данные для началной инициализации БД

В проекте реализована функциональность 
- вывоб в виде таблице информации о клиентах физ лицах и юр лицах
- Добавление новых клиентов с помощью ORM и с обычным sql запросом
- Различные linq запросы
